# **1. XP (Experience Points)**

**XP adalah mata uang progres yang didapatkan saat user menghafal. XP tidak bisa dibeli dan hanya mencerminkan effort nyata — tidak dapat dikonversi ke GEMS**

## **Cara Mendapatkan XP**

- **1 step hafalan jika benar = 4 XP**
- **XP murni mencerminkan progres — tidak bisa dibeli atau dikonversi**
- **1 misi harian = 35 XP**

> XP berbeda dari Gems. Gems energy didapat dari reward streak pekanan, bukan dari XP.

## **Fungsi XP**

- **Leveling dan ranking leaderboard**
- **Menentukan level user (memengaruhi reward Gems per Juz)**

***Catatan Desain: XP sengaja dipisahkan dari Energy supaya tidak ada shortcut — user tidak bisa "membeli" progres.***
