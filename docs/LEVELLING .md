# PROLOGUE

Menghafal Quran perlu sistem. User ditentukan levelnya berdasarkan onboarding yang berfungsi sebagai levelling map.

### Onboard Key Questions:

- Bagaimana experience menghafalnya?
- Bagaimana kualitas bacaannya?

---

# 3 STAGES OF LEVELLING

## Beginner Level (Juz Amma)

Level ini adalah permulaan untuk pengguna yang baru mulai menghafal Quran. Terutama untuk mualaf, awam yang pertama kali menghafal, dll. Target besarnya adalah Juz 30 (Juz Amma), yang kemudian dibagi menjadi mini target.

### Key Ingredients

- As fun as possible
- As easy as possible
- Experimental audio
- Stepnya banyak, kecil-kecil, membiasakan user dengan ayat demi ayat
- Pengenalan menghafal Quran

### Mini Target Juz Amma

| Tangga         | Target Surah         | Reward        |
| -------------- | -------------------- | ------------- |
| Tangga Pertama | An-Naas — Ad-Dhuha   | Checkpoint    |
| Tangga Kedua   | Al-Layl — At-Takwir  | Checkpoint    |
| Tangga Ketiga  | An-Naziat — An-Naba' | Badge per Juz |

---

## Mid Level (Tabarak, Juz 28, Al-Waqi'ah, Ar-Rahman)

Level ini adalah level lanjutan setelah level pertama. Ditujukan bagi user yang sudah pernah menghafal Juz Amma dan ingin melanjutkan hafalannya, atau user yang telah menyelesaikan Beginner Level. Targetnya dibuat 2–3× lebih besar dari Beginner Level, namun tetap dibagi ke dalam mini target.

### Key Ingredients

- As fun as possible
- As easy as possible
- Experimental audio
- Stepnya banyak, kecil-kecil, membiasakan user dengan ayat demi ayat

### Mini Target (Per Juz = 1 Tangga)

#### Juz 29

| Tangga | Target Surah                                      |
| ------ | ------------------------------------------------- |
| 1      | Al-Mulk, Al-Qalam, Al-Haqqah                      |
| 2      | Al-Ma'arij, Nuh, Al-Jinn, Al-Muzzammil            |
| 3      | Al-Muddatstsir, Al-Qiyamah, Al-Insan, Al-Mursalat |

#### Juz 28

| ---------- | -------------------------------------------------------- |

| Ar-Rahman  | Boleh diambil di Mid Level atau disimpan untuk Pro Level |

---

## Pro Level (Juz 1–27)

Level ini ditujukan bagi user yang telah menyelesaikan Mid Level atau user yang sudah memiliki hafalan sebelumnya dan ingin melanjutkan secara sistematis. Target besarnya adalah Juz 1–27. Dengan target yang begitu besar, Pro Level tetap memiliki mini target.

User yang memilih Pro Level memiliki intensi yang lebih kuat. Dengan asumsi telah lebih dulu berpengalaman dan memiliki hafalan, maka learning path yang diberikan lebih serius — tapi tetap fun, insyaAllah.

### Key Ingredients

- Discipline over fun
- Audio-visual experience
- More focus, new learning path
- Badge per surah dan per juz

### Mini Target (Per Juz – Per 5 Halaman)

Berikut adalah struktur mini target untuk setiap juz dari Juz 1 hingga Juz 27:

**Setiap Juz dibagi menjadi 4 bagian:**

| Bagian   | Halaman   |
| -------- | --------- |
| Bagian 1 | Hal 1–5   |
| Bagian 2 | Hal 6–10  |
| Bagian 3 | Hal 11–15 |
| Bagian 4 | Hal 16–20 |

> Pola ini berulang untuk **Juz 1 sampai Juz 27**.

#### Contoh Penerapan

**Juz 1**

- Hal 1–5
- Hal 6–10
- Hal 11–15
- Hal 16–20

**Juz 2**

- Hal 1–5
- Hal 6–10
- Hal 11–15
- Hal 16–20

_(Dan seterusnya hingga Juz 27 dengan pola yang sama)_

| Level    | Target Utama                 | Mini Target                    | Key Ingredient Utama             |
| -------- | ---------------------------- | ------------------------------ | -------------------------------- |

| Pro      | Juz 1–27                     | Per juz = 4 bagian (5 halaman) | Disiplin, audio-visual, badge    |
